## NFS Minimap Maker.
<img align="right" width="128" height="128" alt="NFSMinimapMaker" src="https://github.com/user-attachments/assets/fb024a27-9a16-4943-83ba-e00c25b25316" />
With this tool, you no longer have to spend 5, 10, or even 20 minutes building a single minimap manually.  

### Features:
* Supports output sizes from **1024** up to **8192**.
* Currently supports: NFSMW05.
* Automatically generates a properly named minimap file.
* Optimized workflow with Fast and Slow Modes.
* Compatible with PNG, JPG, JPEG, and DDS formats.

### Planning:
* Support for NFS Underground 2, ??.
* Batch creation of multiple minimaps at once.
* Advanced texture settings with a built-in TPKBlocks attribute configurator.
