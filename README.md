## NFS Minimap Maker.
Making life easier for modders, you no longer need to spend 5 or even 20 minutes creating a single minimap.  
Although the program is small, it makes your work much easier, especially manual work.  

### What can it do?  
* Supports sizes from 1024 to 8192.
* Support Games: Only NFS Most Wanted.
* Defined minimap name.
* Maximum optimization of work to creation (Like Modes: Fast and Slow whoops).
* Supported formats: png, jpg, jpeg, dds.

### Planning:
* Support for: NFS Underground 2, ??.
* Creating multiple minimaps at once.
* Flexible settings for the textures (Inside TPKBlocks) attribute configurator.
